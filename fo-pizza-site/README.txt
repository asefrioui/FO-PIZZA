FØ PIZZA — CONCEPT DE SITE PREMIUM

CONTENU
- index.html : structure sémantique et contenu de la page
- styles.css : direction artistique, responsive et effets 3D
- script.js : carte dynamique, studio interactif, commandes et animations
- assets/ : visuels locaux optimisés

OUVRIR LE SITE
Double-cliquez sur index.html, ou lancez un serveur local dans ce dossier :

  python -m http.server 4173

Puis ouvrez http://127.0.0.1:4173/

FONCTIONNALITÉS
- Design premium responsive (desktop, tablette et mobile)
- Hero immersif avec effet de profondeur
- Studio de composition de pizza en CSS 3D
- Carte dynamique pizzas / desserts
- Sélecteur Paris 13 / Saclay
- Fenêtre de commande reliée aux parcours officiels
- Navigation mobile, animations au défilement et accessibilité clavier
- Respect de prefers-reduced-motion
- Photos WebP optimisées pour un chargement rapide

SOURCES DE CONTENU
- Adresses, téléphones, prix et liens : site officiel fo-pizza.com
- Photos des restaurants : médiathèque du site officiel FØ Pizza
- Visuel héro : création originale générée pour cette maquette
